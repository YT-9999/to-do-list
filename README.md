# 任务助手

一个极简风格的任务拆分和任务清单工具，特别适合ADHD用户使用。

## 功能特点

### 任务拆分
- 输入复杂任务，自动拆分成多个易执行的步骤
- 调用DeepSeek API进行智能拆分
- 可以勾选完成每个步骤
- 支持重新开始

### 任务清单
- 添加每日待办事项
- 完成任务后显示鼓励语和名言
- 自动按日期分组
- 支持删除任务

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件
2. 首次使用时，会弹出API密钥设置窗口
3. 输入你的DeepSeek API密钥（从 https://platform.deepseek.com/ 获取）
4. 开始使用任务拆分和任务清单功能

## 技术栈

- 纯HTML/CSS/JavaScript
- 无需构建工具，直接运行
- 使用localStorage进行数据持久化
- 调用DeepSeek API进行任务拆分

## 设计理念

- 极简风格，参考纽约时报设计
- 移动端友好，响应式布局
- 减少视觉干扰，专注任务完成
- 提供即时正反馈
