# 任务助手 (Task Assistant)

一个专为ADHD用户和拖延症患者设计的智能任务管理工具，通过AI技术将复杂任务拆分为简单易执行的步骤，提供情绪支持和即时正反馈。

## 核心功能

### 1. 智能任务拆分
- **功能**：输入复杂任务，自动拆分成5-15个极其简单的最小物理动作步骤
- **技术**：调用DeepSeek API进行智能拆分，支持缓存机制提高响应速度
- **应用场景**：特别适合ADHD用户面对复杂任务时的执行困难
- **特点**：步骤细到"掀开被子"、"坐起来"这种具体程度，从最简单的动作开始

### 2. 任务清单管理
- **功能**：添加每日待办事项，完成后显示鼓励语和名言
- **技术**：使用localStorage本地存储，自动按日期分组
- **特点**：提供即时正反馈，支持删除任务，响应式设计

### 3. 情绪记录与分析
- **功能**：记录每日情绪状态，自动分析情绪倾向
- **技术**：调用AI进行情绪分析，支持缓存机制
- **特点**：当检测到负面情绪时，主动提供心理支持

### 4. AI心理医生
- **功能**：提供基于AI的心理咨询服务，帮助缓解负面情绪
- **技术**：调用DeepSeek API，优化prompt提高响应质量
- **特点**：提供专业的心理支持，帮助用户应对拖延和情绪问题

## 技术特点

### 1. 性能优化
- **缓存机制**：实现24小时本地缓存，减少重复API调用
- **超时处理**：API请求超时设置为90秒，确保获得结果
- **重试机制**：网络错误自动重试，提高成功率
- **参数优化**：减少max_tokens，提高响应速度

### 2. 用户体验
- **加载状态**：所有AI功能都有明确的加载指示器
- **错误处理**：友好的错误提示，避免用户困惑
- **极简设计**：参考纽约时报设计风格，减少视觉干扰
- **移动端友好**：响应式布局，支持手机和平板使用

### 3. 技术栈
- **前端**：纯HTML/CSS/JavaScript
- **存储**：localStorage本地存储
- **API**：DeepSeek API（支持OpenAI等其他API）
- **部署**：GitHub Pages静态部署

## 使用方法

### 首次使用
1. **打开应用**：直接在浏览器中打开 `index.html` 文件
2. **设置API密钥**：首次使用时，会弹出API密钥设置窗口
3. **输入密钥**：输入你的DeepSeek API密钥（从 https://platform.deepseek.com/ 获取）
4. **开始使用**：设置完成后，即可开始使用所有功能

### 任务拆分使用
1. 在"任务拆分"输入框中输入复杂任务
2. 点击"拆分"按钮
3. 等待AI智能拆分完成（通常10-30秒）
4. 按照拆分的步骤逐步执行，完成后勾选

### 任务清单使用
1. 在"任务清单"输入框中添加每日待办
2. 完成任务后点击任务文本，会显示鼓励语
3. 鼠标悬停任务会显示删除按钮

### 情绪记录使用
1. 在"情绪记录"输入框中描述当前情绪
2. 点击"保存"按钮
3. 系统会自动分析情绪，并在负面情绪时提供心理支持

### AI心理医生使用
1. 当情绪记录显示负面情绪时，会自动弹出聊天窗口
2. 也可以通过右上角按钮手动打开
3. 与AI心理医生交流，获取心理支持和建议
